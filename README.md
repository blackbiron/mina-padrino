## Description

[Mina][mina] tasks for deploying [Padrino][padrino] apps.

## Features

* Sets `PADRINO_ENV=`
* Supports `db:migrate`

## Examples

    require 'mina/padrino'

## Requirements

* [mina] ~> 0.3

## Install

    $ gem install mina-padrino
